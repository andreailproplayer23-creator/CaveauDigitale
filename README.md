# 🛡️ Caveau Digitale v2.5

[![Python Version](https://img.shields.io/badge/python-3.13-blue.svg)](https://www.python.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Framework](https://img.shields.io/badge/UI-CustomTkinter-orange.svg)](https://github.com/TomSchimansky/CustomTkinter)

**Caveau Digitale** è un'applicazione desktop sicura e moderna progettata per proteggere i tuoi file e cartelle personali tramite crittografia di livello bancario (AES). Gestisci i tuoi dati sensibili con un'interfaccia intuitiva e non temere mai più di perdere l'accesso grazie al doppio sistema di recupero.

---

## ✨ Caratteristiche Principali

* **🔐 Crittografia Militare**: Utilizza la libreria `cryptography` per garantire che i tuoi file siano inaccessibili senza la chiave corretta.
* **📂 Supporto File & Cartelle**: Trascina e rilascia singoli file o intere directory (vengono compresse e criptate automaticamente).
* **🔑 Gestione Password Flessibile**: 
    * Usa la **Master Password** per una gestione rapida.
    * Imposta **Password Singole** per file estremamente sensibili.
* **🧠 Sistema di Recupero Avanzato**: 
    * Domande di sicurezza per la Master Password.
    * Domande personalizzate per ogni singolo file criptato con password dedicata.
* **🌍 Multi-lingua**: Interfaccia disponibile in **Italiano**, **Inglese**, **Spagnolo** e **Francese**.
* **🔍 Ricerca Intelligente**: Trova i tuoi file nell'archivio in tempo reale mentre scrivi.
* **🎨 UI Moderna**: Interfaccia scura (Dark Mode) basata su CustomTkinter.

---

## 🛠️ Installazione (Utenti)

Se non sei un programmatore, puoi scaricare l'installer già pronto:

1.  Vai nella sezione [Releases](https://github.com/andreailproplayer23-creator/CaveauDigitale/releases).
2.  Scarica `CaveauDigitale_Setup.exe`.
3.  Esegui l'installazione e avvia il programma dal Desktop.
